Twerk & Chair

Twerking & Chair Dance Classes

* we are currently not offering this class - sorry!

Learn to twerk and chair dance in this high energy workout class. If you're looking to burn a lot of calories while learning an impressive skill, this class is for you! Each class includes a warm up which will help improve your core strength and flexibility, followed by a combination of twerking and chair dance moves. You may learn a combination or routine.

* Please note this is not a lap dance class. This is a fitness class using the chair as a prop during the warm up, workout and also for dance routines.
